# Security Policy

## Supported Versions
This is a frontend-only demo project. Security fixes are applied to the latest version on the default branch.

## Reporting a Vulnerability
If you discover a security issue, please report it privately:
- Email: jennynjoku148@gmail.com

Please include:
- Steps to reproduce
- Screenshots (if helpful)
- Any suggested fix

## Notes
- This project is a static demo and does not include a backend/database.
- Do not submit real sensitive information (IDs, bank details, passwords).
